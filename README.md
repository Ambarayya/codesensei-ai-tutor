# CodeSensei - AI Tutor

CodeSensei is an intelligent YouTube tutor that:
- Summarizes programming videos based on skill level (Beginner, Intermediate, Advanced)
- Breaks content into timestamped topics with explanations
- Allows topic-wise deep dives and exploration
- Generates quizzes from the video content
- Offers interactive code debugging and practice
